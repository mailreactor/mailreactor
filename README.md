# Mail Reactor

A headless email client with REST API.

## What is Mail Reactor?

Mail Reactor will be a self-hosted, open-source email client that provides REST API access to IMAP/SMTP email accounts. It aims to transform email integration from weeks to minutes for developers.

**Status:** Currently in development (Sprint 0 - Infrastructure Setup)

## Development

This repository is for **Mail Reactor development**, not end-user installation.
